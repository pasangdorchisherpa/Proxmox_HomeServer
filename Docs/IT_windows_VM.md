# Windows Virtual Machine 

## Overview 

This Windows virtual machine runs on Proxmox VE and is used as a **learning environment** for basic IT administration and system management tasks.

It is not intended for production use, but rather as a sandbox for experimentation.

---

## Setup Summary

- Windows VM hosted on Proxmox VE
- Assigned fixed CPU cores and memory
- Accessed through Proxmox console and remote desktop

---

## Primary Use Cases

### IT Administration Practice
- User and permission management
- File and folder access control
- Basic system configuration tasks

### File & Access Management
- Creating users and groups
- Assigning read/write permissions
- Understanding how access control works in Windows environments

### Safe Learning Environment
- Test changes without affecting real systems
- Reset or rebuild VM if needed
- Practice troubleshooting and system maintenance

---

## Use Case in My Home Server

- Hands-on experience with Windows administration
- Learning concepts applicable to help desk and junior IT roles
- Similar to Linux-based services running in the homelab
