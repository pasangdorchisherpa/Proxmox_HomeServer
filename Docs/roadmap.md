# Homelab Roadmap

This document outlines planned additions and improvements to my Proxmox VE homelab.

---

## Short-Term Goals

### WireGuard VPN
- Secure remote access to the home network
- Access internal services without exposing ports publicly

### TrueNAS (NAS VM)
- Centralized file storage
- Backups for VMs and personal data
- Shared storage across devices

### Nextcloud
- Self-hosted file and photo sharing
- Personal cloud replacement
- Access controlled via VPN

---

## Long-Term Goals

- Automated backups
- Monitoring and alerting
- Firewall segmentation
- Infrastructure documentation
- Infrastructure-as-Code experimentation

---

## Overall Goal

Build a secure, scalable, and well-documented homelab to:
- Learn system administration
- Practice networking concepts
- Gain hands-on experience with virtualization and self-hosted services
