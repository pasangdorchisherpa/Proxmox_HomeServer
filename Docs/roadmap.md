# Homelab Roadmap

This document outlines planned additions and improvements to my Proxmox VE homelab.

---

## Long-Term Goals

- Automated backups
- Monitoring and alerting
- Firewall segmentation
- Infrastructure documentation
- Infrastructure-as-Code experimentation

---

## Overall Goal

Build a secure, scalable, and well-documented homelab to:
- Learn system administration
- Practice networking concepts
- Gain hands-on experience with virtualization and self-hosted services
